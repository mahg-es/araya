# Graph Builder Engine

The Graph Builder generates knowledge graph entities and relationships
automatically from existing ARAYA artifacts. No manual maintenance.

## Principle

Humans create intent. Agents create artifacts. ARAYA generates relationships.

## How It Works

```
Artifacts (specs, ACs, capabilities, trajectories, etc.)
       │
       ▼
Mapping Rules (deterministic transforms)
       │
       ▼
Entities + Relationships (graph-ready data)
       │
       ▼
Organizational Knowledge Graph (Batch 9)
```
