# Visual Identity

Design and maintain visual identity systems — logo, color palette, typography,
iconography, and imagery guidelines — creating a recognizable, consistent
brand presence across all touchpoints.

## What problem this solves
A logo alone is not a brand. Visual identity is the system of design decisions
that makes every output recognizable as yours — from business cards to
billboards to browser tabs. This skill designs that system, documented for
anyone creating branded materials.

## When to use
When establishing a new brand. When refreshing an existing brand. When
inconsistency across properties becomes visible. When onboarding new
designers or developers.

## Input
Brand strategy, logo files, design preferences, competitive landscape.

## Output
A visual identity guide with logo usage, color system, typography, iconography,
and application examples.

## Steps
1. Define the brand personality: professional, innovative, trustworthy, creative?
2. Design/refine logo: primary, icon-only, horizontal/vertical, monochrome variants
3. Build color system: primary (1-2), secondary (2-3), neutral (3-5), semantic (3)
4. Select typography: heading font, body font, monospace for code — with fallback stacks
5. Define iconography style: filled, outlined, duotone — consistent across the system
6. Create imagery guidelines: photography style, illustration style, data visualization
7. Produce application examples: website, business card, presentation, social media
8. Package as brand guidelines document in my-brands-guidelines

## Rules
- One visual identity across all properties — brand consistency builds recognition
- Logo: provide SVG (vector), PNG (raster), and favicon (ICO) in multiple sizes
- Color: 1-2 primary, 2-3 secondary, semantic (error/success/warning), neutrals
- Typography: 2 fonts max (heading + body) + monospace — more clutter, not more design
- Spacing and layout consistent with the identity — not an afterthought
- Accessibility: contrast ratios, colorblind-safe alternatives, focus indicators
- Coordinate with Dorcas (brand governance) and brand-compliance (auditing)
